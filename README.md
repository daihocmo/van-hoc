# Chương trình học

Dành cho những bạn có thể học hoàn toàn bằng Tiếng Anh: Đọc: [Chương trình đào tạo Literature bằng Tiếng Anh TẠI ĐÂY](https://hocbigg.github.io/literature/).

## Giáo trình được sử dụng

- [Lí luận Văn học của Lê Lưu Oanh](https://archive.org/details/li-luan-van-hoc/) - Ký hiệu là LLVH
- [Nhập môn Ngôn ngữ học lý thuyết (= Introduction to Theoretical Linguistics)" của John Lyons (Vương Hữu Lễ dịch)](https://archive.org/details/nhapmonngonnguhoclythuyet_gd_1996/) - Ký hiệu là NNHLT
- [Lý luận văn học so sánh - Nguyễn Văn Dân](https://archive.org/details/ly-luan-van-hoc-so-sanh) - Ký hiệu là LLVHSS
